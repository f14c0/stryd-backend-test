#Stryd Backend test

##How to run 
Install dependencies
```bash
pip install -r requirements.txt
```
Run [FastAPI](https://fastapi.tiangolo.com/) Server
```bash
python -m uvicorn main:app 
```
Running tests

```bash
pytest test.py
```
